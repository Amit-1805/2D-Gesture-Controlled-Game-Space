# 🎮 2D Gesture Controlled Game

> Control the game using your **hand gestures** instead of a keyboard or mouse!
> Built with **Python**, **OpenCV**, and **Pygame** to provide a fun, touch-free gaming experience using computer vision.

---

## 📌 Overview

The **2D Gesture Controlled Game** is an interactive computer vision project that enables players to control a basket using **real-time hand gestures** detected through a webcam. The objective is to catch falling objects while avoiding misses to achieve the highest possible score.

This project demonstrates the integration of **computer vision** and **game development**, showcasing how natural hand movements can be used as an intuitive game controller.

---

## ✨ Features

* 🖐️ Real-time hand gesture recognition
* 📷 Webcam-based control
* 🎯 Smooth basket movement
* 🍎 Falling object mechanics
* 💥 Collision detection
* 🏆 Live score tracking
* ❤️ Life/Game Over system
* 🔄 Restart functionality
* ⚡ Lightweight and beginner-friendly

## 🚀 Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/2D-Gesture-Controlled-Game.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd 2D-Gesture-Controlled-Game
```

### 3️⃣ Install Dependencies

```bash
pip install pygame opencv-python numpy
```

Or

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Game

```bash
python main.py
```

## 🎮 How to Play

1. Launch the game.
2. Allow webcam access.
3. Move your hand left or right.
4. The basket follows your hand movement.
5. Catch the falling objects to earn points.
6. Avoid missing too many objects.
7. Try to achieve the highest score!
